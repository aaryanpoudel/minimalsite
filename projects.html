Bus Tracking and Reservation System

Bus Tracking and Reservation System, is a real-time mobile application used to view the current location of buses around our locality. There are two apps; one on the user side and the other on the driver side. The driver app is used to broadcast the location of the vehicle at real-time. The user app can be used to reserve seats and to get a maps view of the current location of the bus.

Created Using:

Java, Firebase, GoogleMaps API

July, 2019

LPG Gas level Indicator

It's a hardware project that provides infromation about the LPG gas to the user. The project uses a load sensor to calculate the weight of the cylinder. By monitoring the weight, the volume of gas left can be determined. Also, there's some additional gas sensors, which detect the leakage of the gas. A mobile app was created to show all these data to the user and also for providing alerts during the gas leakage.

Created Using:

Arduino, Load sensor, Gas sensor, Java

February, 2018
